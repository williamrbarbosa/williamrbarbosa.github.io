## About

<div align="left">
Full Stack Developer with solid experience building and maintaining web applications using Node.js, React, NestJS, PHP, SQL, and modern frontend/backend architectures.

Over the years, I have worked on projects across different industries such as ERP, education, healthcare, logistics, factories, and sports platforms. This experience helped me develop adaptability, business understanding, and the ability to deliver practical solutions in dynamic environments.

My main focus is building reliable, scalable, and maintainable systems while collaborating closely with product, design, and engineering teams.

I have hands-on experience with REST APIs, database design, integrations, bug fixing, performance improvements, and delivering features end-to-end.

Technologies I work with include Node.js, NestJS, React, Next.js, Angular, PHP, PostgreSQL, MySQL, Docker, AWS, Redis, RabbitMQ, and CI/CD workflows.

I’m always looking to grow technically, contribute to strong teams, and build products that create real impact.
